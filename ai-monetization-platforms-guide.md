## 1. Gumroad — 数字产品销售

### 平台简介
Gumroad 是目前最流行的数字产品销售平台，适合卖代码、模板、电子书、prompt 库、音乐、设计素材等。操作简单，抽成 10%（有涨价消息）。

### 注册步骤
1. 访问 [gumroad.com](https://gumroad.com)
2. 点击 "Sign Up"，用 Google 账号或邮箱注册
3. 填写用户名（会成为你的店铺地址，如 `gumroad.com/yourname`）
4. 验证邮箱
5. 设置店铺名称和简介
6. 绑定收款方式（Stripe / PayPal）

### AI 自动化方案
- **产品生成**：用 Claude/DeepSeek 生成代码模板、prompt 包
- **批量生产**：一次生成 10-50 个模板，批量上传
- **文案撰写**：AI 生成产品描述、销售页文案
- **客户服务**：AI 处理常见问题（可设常见问题模板）
- **邮件营销**：Gumroad 内置邮件功能，可 AI 批量发送

### 详细操作教程

#### 第一步：确定产品类型
热门产品类型：
- 代码模板（Astro 主题、Turborepo 配置）
- AI 提示词包（SEO 博客、外贸开发信）
- Notion 模板（项目管理、博客运营）
- 电子书（使用指南、教程）
- 设计素材（图标、UI 套件）

#### 第二步：用 AI 生成产品
以"AI 提示词包"为例：
```
用 Claude 生成 50 个针对小红书美妆品牌的推广提示词，
每个提示词包含：适用场景、目标受众、核心要点
```

#### 第三步：打包产品
1. 整理成 PDF 或 ZIP
2. 写好 README 说明
3. 设计封面（用 Canva / Figma）

#### 第四步：上传产品
1. 点击 "New Product"
2. 填写标题、描述
3. 上传文件
4. 设置价格（可设分级：$9 / $19 / $49）
5. 添加封面图
6. 提交发布

#### 第五步：推广
- 在 Twitter/X、Reddit、LinkedIn 分享链接
- 加入相关社群推广
- 写博客介绍产品

### 变现技巧
- 设置 "Pay what you want"（让用户自己定价）
- 捆绑销售：多个产品打包卖
- 订阅模式：每月更新产品
- Gumroad Discover 流量（平台内置搜索）

### 常见问题
- **收款**：Stripe 绑定后自动结算
- **抽成**：每笔 10%（+ Stripe 手续费约 3%）
- **文件大小限制**：免费版单文件最大 100MB

## 2. Lemon Squeezed — 低抽成数字商店

### 平台简介
Lemon Squeezed 是 Gumroad 的竞争者，抽成更低（5-8%），支持订阅制，适合想要更低抽成的卖家。

### 注册步骤
1. 访问 [lemonsqueezy.com](https://www.lemonsqueezy.com)
2. 点击 "Get started"
3. 用 Google 账号或邮箱注册
4. 填写店铺信息
5. 绑定 Stripe 账户
6. 完成入驻审核

### 与 Gumroad 对比
| 特性 | Gumroad | Lemon Squeezed |
|------|---------|----------------|
| 抽成 | 10% | 5-8% |
| 订阅支持 | 有 | 有（更完善） |
| 退款政策 | 宽松 | 严格 |
| 平台费用 | 有隐藏费用 | 更透明 |

### AI 自动化方案
与 Gumroad 类似：
- AI 生成数字产品
- AI 批量处理订单
- AI 客户支持
- AI 营销邮件

### 详细操作教程
1. 创建账户并完成 Stripe 绑定
2. 点击 "New Product"
3. 填写产品详情（与 Gumroad 类似）
4. 设置价格，可选订阅模式
5. 上传文件
6. 获取销售链接

### 变现技巧
- Lemon Squeezy 的 SEO 比 Gumroad 差，需要自己引流
- 适合订阅制产品（如每月更新的模板包）

## 3. Koji — 移动端数字商店

### 平台简介
Koji 是一个移动端优化的数字产品平台，类似于 Linktree 但带商店功能。适合在 Instagram、TikTok 等平台引流。

### 注册步骤
1. 下载 Koji App（iOS/Android）或访问 [koji.com](https://koji.com)
2. 用 Google 或 Apple ID 注册
3. 选择用户名
4. 创建第一个"商品"

### 独特功能
- **Link in Bio**：可作为 Linktree 替代品
- **内嵌购买**：别人可直接在你的链接里付款
- **Discord 集成**：自动发放数字商品到 Discord

### AI 自动化方案
- AI 生成商品描述
- AI 批量创建多个商品
- AI 回复消息

### 详细操作教程
1. 创建 Koji 主页
2. 添加商品类型（下载、打赏、订阅）
3. 设置价格
4. 分享你的 Koji 链接到社交媒体
5. 商品售出后，Koji 自动发放

### 适合卖什么
- 社交媒体粉丝变现
- 小额打赏
- 虚拟礼物
- 小额数字产品（$5-$20）

## 4. Envato / ThemeForest — 代码模板市场

### 平台简介
Envato 是全球最大的数字资产市场，其中 ThemeForest 卖网站主题，CodeCanyon 卖代码插件。适合开发者。

### 注册步骤
1. 访问 [themeforest.net](https://themeforest.net)
2. 点击 "Create Account"
3. 填写个人信息（注意填真实信息）
4. 验证邮箱
5. 提交首个产品等待审核

### 抽成和收入
- 作者收入：作者拿 50-70%（按销售额递增）
- Envato 拿 30-50%
- 最低定价建议：网站主题 $29 起

### AI 自动化方案
- **代码生成**：Claude Code 生成 WordPress 主题 / Astro 模板
- **UI 设计**：AI 生成设计稿
- **文档撰写**：AI 生成使用文档
- **响应式测试**：AI 检验多端显示

### 详细操作教程

#### 第一步：准备产品
1. 确定主题类型（WordPress、Astro、HTML5）
2. 用 Claude Code 生成完整代码
3. 包含：
   - 主题文件
   - 文档（README.md）
   - 演示数据
   - 授权文件

#### 第二步：创建 Envato 账户
1. 访问 ThemeForest/CodeCanyon
2. 完成注册和身份验证
3. 设置 Payoneer 或 PayPal 收款

#### 第三步：提交产品
1. 进入 "Author Dashboard"
2. 点击 "New Item"
3. 选择类别
4. 上传压缩包（ZIP 格式）
5. 填写：
   - 标题（SEO 优化）
   - 描述
   - 标签
   - 截图
6. 提交审核

#### 第四步：等待审核
- 首次审核约 7-10 天
- Envato 对质量要求很高
- 需要符合他们的编码标准

### 变现技巧
- 参考畅销产品的设计和功能
- 覆盖热门分类（WordPress 主题需求大）
- 持续更新版本
- 提供技术支持

### 常见问题
- **审核严格**：代码质量、文档、演示都有要求
- **等待时间长**：审核可能需要多轮
- **收入结算**：每月 15 日结算，需达最低额度

## 5. Patreon — 创作者订阅平台

### 平台简介
Patreon 是订阅制平台，粉丝每月付费获取独家内容。适合持续输出内容的创作者。

### 注册步骤
1. 访问 [patreon.com](https://www.patreon.com)
2. 点击 "Create on Patreon"
3. 用邮箱或 Google 注册
4. 填写创作者信息：
   - 账号名称
   - 简介
   - 主要内容类型
5. 设置订阅层级
6. 连接 Stripe 账户
7. 提交审核（一般 1-2 天）

### 订阅层级设置建议
| 层级 | 价格 | 内容 |
|------|------|------|
| 免费 | $0 | 公开内容 |
| 基础 | $3-5/月 | 粉丝专属内容 |
| 中级 | $10-15/月 | + 社区 + 早期内容 |
| 高级 | $20+/月 | + 1v1 咨询 + 定制内容 |

### AI 自动化方案
- **内容生成**：Claude/DeepSeek 批量生成文章/视频脚本
- **选题研究**：AI 分析趋势挖掘话题
- **多语言翻译**：AI 翻译内容吸引海外用户
- **社群运营**：AI 回答常见问题
- **邮件自动化**：Patron 邮件自动发送

### 详细操作教程

#### 第一步：确定定位
热门领域：
- 游戏、动漫、创意艺术
- 技术教程、编程课程
- 生活、健身、美食
- 时事、行业洞察

#### 第二步：创建 Patron 内容
1. 设置每月内容计划（如：每周 3 篇）
2. 用 AI 生成内容大纲
3. 人工审核/修改
4. 发布

#### 第三步：设置订阅层级
1. 点击 "Edit tiers"
2. 添加层级（建议 3-5 个）
3. 每个层级设置：
   - 价格
   - 标题（如"铁粉"）
   - 描述（包含哪些权益）
   - 封面图

#### 第四步：推广 Patreon
1. 在其他平台放 Patreon 链接
2. 设置"限时加入"激励
3. 与其他创作者合作推广
4. 用 AI 生成推广文案

### 变现技巧
- **内容差异化**：提供其他地方没有的内容
- **早期福利**：早期粉丝给额外福利
- **社区建设**：建立 Discord/Slack 社区
- **里程碑**：达成目标后解锁特别内容

### 常见问题
- **最低收入**：需达 $100 才能提现
- **平台抽成**：Patreon 抽 5-12%
- **审核**：某些内容类别受限

## 6. Substack — Newsletter 付费订阅

### 平台简介
Substack 是 Newsletter 订阅平台，专注于文字内容。免费版可发邮件，付费版可设置订阅墙。

### 注册步骤
1. 访问 [substack.com](https://substack.com)
2. 用邮箱注册
3. 设置账户名（成为 substack.com/@yourname）
4. 创建出版物
5. 设置发布频率
6. 绑定 Stripe 收款

### 与 Patreon 对比
| 特性 | Patreon | Substack |
|------|---------|----------|
| 形式 | 订阅+社区 | 邮件 newsletter |
| 适合内容 | 视频/社区 | 深度文章 |
| 平台费用 | 5-12% | 10%（付费 newsletters） |
| 用户关系 | 更强 | 较弱 |

### AI 自动化方案
- **文章生成**：Claude 生成深度分析文章
- **选题研究**：MiniMax 搜索热点
- **邮件优化**：AI 生成邮件标题（提高打开率）
- **翻译分发**：AI 翻译扩展到英文市场

### 详细操作教程

#### 第一步：确定 Newsletter 主题
- 垂直领域（AI、科技、减肥）
- 热点追踪（行业新闻）
- 个人 IP（你的专业领域）

#### 第二步：设置付费订阅
1. 进入 "Settings" > "Paid newsletters"
2. 点击 "Enable paid subscriptions"
3. 设置价格（建议 $8-$15/月）
4. 可选：
   - 免费内容（引流）
   - 付费全部内容
   - 混合模式

#### 第三步：创建内容
1. 用 AI 分析热点
2. AI 生成文章
3. 人工润色（加入个人见解）
4. 在 Substack 编辑器发布
5. 邮件自动发给订阅者

#### 第四步：增长订阅者
- 在 Twitter/X、LinkedIn 推广
- 与其他 Newsletter 互换推广
- 提供免费样品内容
- SEO 让内容被搜索到

### 变现技巧
- **免费+付费混合**：免费内容吸引，付费内容变现
- **年度订阅折扣**：年付享折扣
- **会员专享活动**：直播、AMA
- **广告**：达到一定规模可接广告

### AI 自动化方案
