在法国创业，第一步就是选择公司形式。根据我的经验，详细对比几种常见选择。

## 常见公司形式

### 1. EI (Entreprise Individuelle) 个体企业

**适合**：自由职业者、小规模经营

**优点**：
- 注册简单，费用低
- 无最低资本要求
- 老板和企业不分离

**缺点**：
- 承担无限责任
- 不能引入投资
- 税务优化空间小

### 2. micro-entreprise 微型企业家

**适合**：初创、兼职、副业

**优点**：
- 手续极简
- 税负低
- 无需会计做账

**缺点**：
- 年营业额有限制（77k以下）
- 不能抵扣很多支出

### 3. SARL 有限责任公司

**适合**：中小企业，家族企业

**优点**：
- 责任有限
- 税务灵活
- 可享受企业主薪资抵扣

**缺点**：
- 最低资本1欧元
- 股东限制2-50人
- 管理相对复杂

### 4. SAS 简易股份公司（最推荐）

**适合**：科技创业、引入投资

**优点**：
- 股份转让灵活
- 可定制股权结构
- 便于上市或被收购
- 创始人保护较好

**缺点**：
- 必须有会计做账
- 成本略高

## 注册流程

### 准备材料

1. 护照复印件
2. 法国居留证
3. 公司地址证明
4. 商业计划书（部分情况）
5. 银行资金证明

### 注册步骤

| 步骤 | 说明 | 耗时 |
|------|------|------|
| 1. 核准公司名称 | online | 1天 |
| 2. 起草公司章程 | 律师或自己用模板 | 1-3天 |
| 3. 验资 | 银行开户存款 | 1周 |
| 4. 报纸公告 | 必须的法定程序 | 1周 |
| 5. 网上注册 | guichet-entreprise | 1-3天 |
| 6. 获得SIRET | 相当于营业执照 | 1周 |

### 总费用

| 项目 | 费用 |
|------|------|
| 公司注册 | 0-500欧元 |
| 律师起草章程 | 500-1500欧元 |
| 报纸公告 | 约200欧元 |
| 会计年费 | 1200-3000欧元 |

## 税务对比

### 2024年主要税种

| 税种 | EI/micro | SARL | SAS |
|------|----------|------|-----|
| 所得税 | 个人所得税 | 按利润征收 | 按利润征收 |
| CFE | 有 | 有 | 有 |
| TVA | 有门槛 | 必须 | 必须 |

### 利润分配税务

```
SARL: 利润 → 公司税(25%) → 分红税(30%)
SAS:  利润 → 公司税(25%) → 分红税(30%)
```

## 我的建议

1. **个人副业**：先选 micro-entreprise
2. **正式创业**：推荐 SAS
3. **自由职业者**：EI 或 micro-entreprise
4. **有投资计划**：必须选 SAS

## 常见问题

**Q: 外国人可以注册吗？**
A: 可以，需要法国居留或创业签证。

**Q: 需要法语好吗？**
A: 官方流程可以用英语，但日常生活中法语更方便。

**Q: 会计费用多少？**
A: 取决于公司复杂度，月费100-300欧元不等。
