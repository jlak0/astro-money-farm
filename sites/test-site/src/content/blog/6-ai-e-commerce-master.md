## 1. AI选品

### 工具使用

- ChatGPT分析市场趋势
- Jungle Scout + AI辅助分析
- 谷歌趋势验证

### 具体方法

```markdown
1. 用ChatGPT列出细分市场机会
2. 用工具验证搜索量和竞争度
3. AI分析评论痛点
4. 快速测试小批量
```

## 2. AI主图设计

### 工具组合

- Midjourney生成场景图
- Remove.bg处理背景
- Canva AI排版

### 高转化主图公式

```
AI生成场景图 + 产品白底图 + 卖点文字 = 高点击率主图
```

### 实战案例

一款保温杯：之前月销200件，用AI重新设计主图后，月销突破2000件。
